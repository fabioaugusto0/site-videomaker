Solicitação de Exclusão de Dados

Esta página informa como solicitar a exclusão de dados pessoais relacionados a Fábio Augusto Videomaker.

Como Solicitar

Envie um e-mail para fabioaugustovideomaker@gmail.com com o assunto Exclusão de dados.

O Que Informar

Inclua seu nome, e-mail, telefone, perfil de rede social ou outro dado usado no contato, para que possamos localizar corretamente as informações relacionadas a você.

Prazo de Atendimento

Após o recebimento da solicitação, analisaremos e removeremos os dados aplicáveis em até 15 dias, salvo quando houver obrigação legal de conservação.

Dados que Podem Permanecer

A exclusão pode não remover registros anonimizados, dados agregados de campanhas, métricas sem identificação pessoal ou informações que precisem ser mantidas por exigência legal.

Contato

Para dúvidas sobre este processo, envie e-mail para fabioaugustovideomaker@gmail.com.
